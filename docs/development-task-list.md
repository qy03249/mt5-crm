# MT5 CRM 开发任务清单

本文档按 Python FastAPI + MySQL 8 技术路线拆分开发任务。任务顺序以“先可运行、再闭环、后联调”为原则。

已确认的第一版范围：

- 按 `docs/mt5-crm-feature-inventory.md` 已梳理菜单开发。
- 暂不做销售线索。
- 暂不做风控中心。
- 暂不做客户前台。
- 暂不做代理前台。
- 前端功能接近参考系统，界面做得更干净。

配置原则：

- 系统名称、Logo、favicon、默认语言、多语言列表通过后台配置。
- 入出金规则通过后台配置。
- 支付/收款方式通过后台配置。
- MT5 服务器、组别、杠杆等通过后台配置。
- 本地数据库连接通过 `.env` 配置，不提交真实密码。

开发时字段依据：

- 字段级数据字典：`docs/data-field-map.md`
- 页面功能盘点：`docs/mt5-crm-feature-inventory.md`
- 技术路线：`docs/python-mysql-technical-route.md`

每个模块开始开发前，先查看 `docs/data-field-map.md` 中对应章节，确认数据库字段、接口字段和页面展示字段。
