# MT5 CRM 功能清单

本文档基于 `https://admin.crmdemopro.cc` 后台系统的只读菜单盘点整理，用于后续开发 MT5 CRM 管理系统时作为功能范围参考。

## 1. 系统定位

该系统属于 MT5 Broker/白标后台管理系统，核心能力包括：

- CRM 客户管理
- MT5 账户管理
- 认证、开户、入金、出金审核
- 财务、交易、佣金报表
- 邮件、短信、操作日志记录
- 平台、MT5、出入金、通知、权限等系统配置

系统重点不是普通销售 CRM，而是围绕 MT5 账户、资金审核、交易数据、代理层级和返佣统计建立的运营后台。

## 14. 开发注意事项

- 所有资金类操作必须有审核状态和操作日志。
- 所有 MT5 写操作必须支持幂等，避免重复入金或重复扣款。
- 出金审核建议支持多级审批。
- MT5 同步任务需要记录同步时间、失败原因和重试次数。
- 报表数据建议使用异步聚合，避免每次查询都实时扫交易表。
- 权限需要区分菜单权限、按钮权限和数据范围权限。
- 客户上下级关系需要支持代理/IB 层级。
- 邮件、短信发送需要记录模板、接收人、状态、失败原因。
- 敏感配置如 MT5 令牌、SMTP 密码、支付密钥需要加密存储。
- 生产环境必须限制后台登录 IP、启用二次验证和审计日志。
